# AI Audit Demo

A standalone audit deck experience built with Next.js.

## Overview

This is a standalone presentation deck for the ThreePoint AI Audit Workshop 2: *Designing the Workflows*.

It covers:
- **Section 01 — How AI actually works**: RAG framework, the agentic loop brain model
- **Section 02 — Logistics & Planning**: Stage 10 deep-dive, brain canvas workflow exercise  
- **Section 03 — Your tasks, your tools**: Individual mapping + Monday decision gates

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — then click "Open Deck" or go straight to [http://localhost:3000/deck](http://localhost:3000/deck).

## Keyboard Controls

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `G` | Slide sorter |
| `Esc` | Close sorter |
| `F` | Fullscreen |
| `Home` / `End` | First / Last slide |

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- No external UI libraries — all custom CSS
- DM Sans + DM Serif Display fonts

## Deployment

Deployed to Vercel. Push to `main` to auto-deploy.
